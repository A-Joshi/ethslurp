Welcome to EthSlurp.

Given the address of any Ethereum account or smart contract 'ethslurp' returns a list of all transactions either to or from that account. For example, 

ethslurp 0xbb9bc244d798123fde783fcc1c72d3bb8c189413

returns all transactions to and from "The DAO." Try it out.

You will notice that the format of the returned data is difficult to read. Use the -p option to pretty print the output. Try this: 

ethslurp -p 0xbb9bc244d798123fde783fcc1c72d3bb8c189413
There are many other options detailed in the documentation.

What Can I Use It For

Let's say you're a gambler and want to participate in an Ethereum gambling website. The website claims to pay out 99% of incoming revenue. Use 'ethslurp' with the -ti (total income) and the -te (total expended) options to check up on them.

Another use might be to keep track of your own smart contract. You can get a report on all the gas spent by your contract, all its income, and any expenditures. Wouldn't it be nice to know if the contract was making money?

The best use we've found for 'ethslurp' is to run in the background of our website in order to prepare block chain data customized to our needs. We run the DAO Reporter website where we present personalized tracking of your ether's interactions with various contracts and other accounts. Pay us a visit.

Building the software

We do not provide binaries (mostly because we don't know how). You have to download the source code and compile it. Don't worry--it's easy. Information on building is provided on the documentation page.

We also don't know how to make a full-fledged package and there are a few dependencies which may not resolve. Please email use with questions: ethslurp@greathill.com
